# Chef

Cookbook on Visionect e-ink device. 

## Setup chef parametes using  Enviorment variables

 |   variable   | default value |
 |--------------|---------------|
 |  DB_USER     | chef          |
 |  DB_NAME     | cookbook      |
 |  DB_PASS     | chef          |
 |  DB_SSLMODE  | disable       |
 |  DB_HOST     | localhost     |
 |  DB_PORT     | 5432          |
 |  LISTEN_ADDR | :4006         |
