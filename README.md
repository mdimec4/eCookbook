# Chef

Cookbook on Visionect e-ink device. 

## Setup chef parametes using  Enviorment variables

 | variable         | default value |
 |------------------|---------------|
 | CHEF_DB_USER     | chef          |
 | CHEF_DB_NAME     | cookbook      |
 | CHEF_DB_PASS     | chef          |
 | CHEF_DB_SSLMODE  | disable       |
 | CHEF_DB_HOST     | localhost     |
 | CHEF_DB_PORT     | 5432          |
 | CHEF_LISTEN_ADDR | :4006         |
